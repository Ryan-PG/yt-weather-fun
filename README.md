# Weather Fun

Weather Fun is a Next.js app that suggests fun activities tailored to your local weather, powered by AI and Open-Meteo APIs. Users can search for their city, view current weather details, and get creative activity ideas generated by AI for their location and conditions.

## Website Preview
![Weather Fun Preview](preview.png)

## Prerequisites
Before running the app, copy `.env.example` and rename it to `.env.local`. Fill in the required environment variables. Alternatively, set the environment variables in your production environment before starting the app.

## Features
- Search for any city using the location search bar
- View real-time weather data (temperature, day/night, etc.)
- Get 5 AI-generated fun activities based on your weather and location
- Beautiful, animated UI with glassmorphism and playful design

## How to Use
1. Run the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   # or
   bun dev
   ```
2. Open [http://localhost:3000](http://localhost:3000) in your browser.
3. Search for your city in the search bar.
4. View the current weather and enjoy AI-suggested activities!
